Switching from iTerm to HyperTerm

23 Jul, 2016

Recently zeit.co released to the world a wonderful terminal built with JavaScript / HTML / CSS called HyperTerm. Even better yet, this terminal is fully customizable with open source plugins. I’m not one to change my development environment constantly, but this electron app was simple to install on OSX and even easier to customize programmatically. My hope is to demo how simple it is to get started with HyperTerm, and show how I only added a few lines to the configuration file to have a terminal I enjoy to use.
